存款帮助

一、公司入款:（赠送存款金额1%手续费）强烈推荐使用


1. 会员登入后点击 [在线存款]，选择 [公司入款]
2. 请选择欲使用的银行卡。
3. 选择银行后，网页会显示可存入的银行账号。可直接点击网页上银行标志，自动为您带到银行首页，
登入您个入网银后，请将款项转入公司提供的入款账号。
※如您使用农业银行/工商银行， 请将公司入款订单号贴入您网银 [备注/附言] 字段
※建议选择与您转账的银行同一家，同银行间转账可以立即到帐，若跨行转账有可能较晚到帐。※

可以自由选择:
(1).网络银行: 登入自己的网络银行页面，从银行网页上转账到指定银行账户 中。
(2).ATM自动柜员机 : 到实体自动柜员机以银行卡或是现金方式存入，没有开启网银功能与存现金的会员可用
(3).银行柜台转账 : 到银行柜台完成转账手续

※公司入款注意事项※
亲切提醒您，公司入款银行随时更变，请于每次入款前，确认您可以使用的入款账号。
如入款账号已过期，美高梅娱乐城恕不负责！万请见谅，感谢配合。

4. 核对提交数据/提交申请
汇款完成请填写并提交相关数据，并备份您的公司入款订单号。 系统在收到款项后会进行比对，如果存款金
额、时间相符合，则会将款项加入您的游戏账号中。即时到账！

5. 若5钟内未到帐，请联系在线客服人员。
客服人员会与您核对存款数据，必要时需要您提供截图、转账数据等相关证明。

二、在线存款 :

1. 会员登入后点击 [在线存款]，选择 [在线付款]
2. 选择入款额度，并请确实填写 ”联络电话” ，如有任何问题，方便美高梅娱乐城客服第一时间与您联系。
3. 选择”支付银行”。
支援借记卡：中国农业银行,中国工商银行,中国建设银行,招商银行,交通银行,上海浦东发展银行,中国光大银行,深圳平安银行。
4. 确认送出后，将请您确认您的支付订单无误，并建议您记录您的支付订单号后，确认送出，并耐心等待加载网络银行页面，传输中已将您账户数据加密，请耐心等待。
5. 进入网络银行页面，请确实填写您银行账号信息，支付成功，额度将在10分钟内系统处理完成，立即加入您的美高梅娱乐城会员账户。

存款需知:
美高梅娱乐城最低存款为$100人民币，最高存款无上限!
未开通网银的会员，请亲洽您的银行柜台办理。
如有任何问题，请洽24小时在线客服。
